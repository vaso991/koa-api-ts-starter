# koa-ts-getting-started

## Create Project:

With NPM:
```
npm create koa-ts-api app-name
```

With Yarn:
```
yarn create koa-ts-api app-name
```

With PNPM:
```
pnpm create koa-ts-api app-name
```


## Setup
```
git clone https://github.com/vaso991/koa-api-ts-starter.git
cd koa-api-ts-starter
npm install
npm run dev
```
